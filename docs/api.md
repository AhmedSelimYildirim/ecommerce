# E-Commerce API

## Endpoints

### User
- POST /api/register
- POST /api/login
- GET /api/users/:id

### Product
- GET /api/products
- GET /api/products/:id
- POST /api/products (admin)
- PUT /api/products/:id (admin)
- DELETE /api/products/:id (admin)

### Cart
- POST /api/cart
- GET /api/cart
- PUT /api/cart/:id
- DELETE /api/cart/:id
